## Summary

PR-1 (this plan) migrates 5 lifecycle ops (`swap`, `stop`, `status`, `list`, `reload-catalog`) from AF_UNIX to NATS via a new `LifecycleMixin` next to `GenerationMixin`. Cross-repo contracts ship on `Roxabi/lyra:staging` first; llmCLI bumps `uv.lock`. Feature flag `LLMCLI_LIFECYCLE_VIA_NATS=1` enables the NATS path during a validation window. Slice 6 (cutover PR — deletes `daemon.py`, flips flag default) gets its own follow-up plan after PR-1 stabilizes.

## Bootstrap Context

From analysis (`artifacts/analyses/34-nats-lifecycle-fold-analysis.mdx`):

- **Shape A locked** (broadcast + app-layer host filter via `_extra_subjects()`, clipool precedent).
- **Drain pattern (A2):** `_draining: asyncio.Event` — set on swap → wait on semaphore drain → execute → clear. Pre-acquire check in `handle()` is atomic in single-threaded asyncio.
- **MRO composition (A3):** `LifecycleMixin._extra_subjects()` returns `[*LIFECYCLE_SUBJECTS, *super()._extra_subjects()]`; `heartbeat_payload()` merges via `super()`.
- **ACL split (O2):** operator nkey gets `pub lyra.llm.lifecycle.>`; worker nkey gets `sub lyra.llm.lifecycle.>`. Inboxes: operator at `_inbox.llm-operator.>`; worker keeps existing `_inbox.hub.>`.
- **Cross-repo ordering:** roxabi-contracts PR merges *first* on `Roxabi/lyra:staging`; llmCLI's `uv.lock` pinned to that SHA in same PR-1.
- **Feature flag:** `LLMCLI_LIFECYCLE_VIA_NATS` in `worker.env`. CLI branches on env var. Slice 6 removes both.
- **CI:** `pytest -m nats` against `nats:2.11-alpine` service container; unit tests stay broker-free.
- **Quadlet:** Exec/HealthCmd verified-aligned via `deploy/entrypoint.sh` (`Exec=llm` → `MODE=llm` → `exec llmcli nats-serve llm ...`).

  ```python
  async def _do_swap(self, msg, req: LifecycleRequest) -> None:
      spec = self._catalog.models.get(req.model_name)
      if spec is None:
          await self._reply_err(msg, req, "llm.lifecycle_rejected", f"unknown model: {req.model_name}", retryable=False); return
      if spec.engine == "remote":
          await self._reply_err(msg, req, "llm.lifecycle_rejected",
              "model uses engine='remote' — managed by LiteLLM proxy", retryable=False); return
      try:
          check_vram_budget(spec, self._catalog.host)
      except ValueError as exc:
          await self._reply_err(msg, req, "llm.lifecycle_rejected", f"vram budget exceeded: {exc}", retryable=False); return

      # Drain (A2): set flag → wait until semaphore is fully released or drain_timeout
      self._draining.set()
      try:
          await asyncio.wait_for(self._wait_sem_idle(), timeout=self._drain_timeout)
      except asyncio.TimeoutError:
          log.warning("lifecycle.swap: drain timeout, hard-cut")
      # Stop-before-start (mirrors daemon._cmd_swap)
      for old_name, old_inst in list(self._instances.items()):
          self._engine_for_spec(self._catalog.models[old_name]).stop(old_inst)
          del self._instances[old_name]
      try:
          new_inst = self._engine_for_spec(spec).start(spec)
      except Exception as exc:  # noqa: BLE001
          await self._reply_err(msg, req, "worker.crash", str(exc), retryable=True); return
      finally:
          self._draining.clear()
      self._instances[req.model_name] = new_inst
      await self._reply_ok(msg, req, data={"model": req.model_name, "port": new_inst.port,
                                            "vram_used_mb": probe_used_vram_mb()})
  ```
